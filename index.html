안녕하세요, 좋은날입니다. SSAFY 14기 대전 캠퍼스 5반 손을준입니다.
🛠️ 2. 주요 기술 스택
  • 백엔드: ASP.NET Core 기반 REST API 개발 (인증·인가, 비동기 처리)
  • 프론트엔드: Blazor를 활용한 SPA 구현 경험 (컴포넌트 구조 설계)
  • ORM:
    • Dapper: 고성능 SQL 매핑 및 복잡한 쿼리 최적화
    • EF Core: 코드 퍼스트 마이그레이션 및 LINQ 활용
  • DB: MySQL을 사용한 데이터 모델링 및 쿼리 최적화 경험
  • 배포: 개인 서버에 Caddy + Docker 구성 후 Dapper 기반 서비스 운영
📁 3. 대표 프로젝트
  • Tus-Blazor-Client
    Blazor 기반 대용량 파일 업로드 클라이언트 라이브러리. Tus 프로토콜을 적용하여 대용량 파일 업로드와 중단 및 재개 기능을 구현 하였음. 또한 해당 오픈소스 프로젝트에 기여되었음.
  • Cloud# (파일 호스팅 서비스)
    ASP.NET Core와 Dapper + EFcore를 사용해 백엔드를 구축하고, Blazor wasm을 통해 사용자 친화적이며 성능도 빠른 파일 관리 기능을 제공.
🏆 4. 주요 성과
  • API 응답 속도 30% 개선 (Dapper 최적화 적용)
  • 대용량 파일 업로드 안정성 강화
  • 기존 파일 호스팅 서비스 대비 파일 처리 성능을 향상시킨 서비스 구축
